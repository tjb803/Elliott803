Documentation Contents

The documentation for the Elliott 803 simulator consists of the following pages:

  • ReadMe. Summary and latest news.
  • History. A brief introduction and history of the Elliot 803 computer.
  • Quick Start. Using the simulator to run a simple program.
  • Operation. More detailed operating instructions.
  • Sample Programs. Some details about the sample programs supplied with the simulator.
  • Telecode. A summary of the 5-hole paper tape character encoding.
  • Utility Applications Details of the utility applications supplied with the simulator.
  • Additional Functions Details for extra function unique to the simulator.
  • Source code details. Some comments and notes about the source code for the simulator.
    Only present if you have downloaded the source code package.
