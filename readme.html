Elliott 803 Simulation

This is a simulation of an Elliott 803 comupter from the early 1960s. The simulation is written entirely in Java and compiled using Java 5, so it should be able to be run on most systems without problems.

Installation

The simulator code, sample programs and documentation are supplied in a single ZIP archive. Download the archive and unzip to a suitable subdirectory on your system.

Running the simulator

To try out the simulator look at the Quick Start guide in the main documentation.

Work In Progress

The following items are either missing from the simulation or could do with some improvement.

The Graphical User Interface

The GUI is fairly simple and not intended to be a completely accurate representation of the original hardware, although it is operated in a broadly similar manner. There's always room for improvement in the GUI.

  • The size and position of windows within the main frame are currently not saved. It is intended to provide an option to save and restore specific machine layouts and configurations.

Unimplemented Hardware

  • The console speaker
  • The film handlers
  • The plotter
  • 8-hole paper tape readers and punches
